Game Overview

Quizard is a quiz game designed for web developers to test their knowledge and skills in various areas of web development. The game features 15 questions that covers the basics of HTML, CSS, and JavaScript.

Game Rules

* Players are presented with a question and four possible answers.

* Players choose an answer, and the game tells them weather it's correct or not.

* Correct answers earns a point, while incorrect answers lose a point.

* Players can not go back to attempted question.

* Wrong answer to question can not be corrected.

* Players earn the sum of #1000 if all 15 questions are answered correctly.

Game Features

* Questions are categorized by topic (HTML, CSS and JavaScript).

* Question bank with 15 questions.
